# System Patterns - Akiriapp

## Design Patterns
- **Theme**: Premium Dark Mode.
- **Color Palette**: 
  - Background: Black (#000000)
  - Accents: Vibrant Turquoise (#0df2df)
  - Text: Deep Blue and White.
- **UI Elements**: 
  - Glassmorphic cards for courses and bottom bars.
  - Highly rounded corners (Full Round/Round Twelve).
  - Horizontal scrolling for categories and popular items.
- **Typography**: Lexend font for a modern, clean look.

## Component Architecture
- Native Android Components (Kotlin).
- XML-based Layouts (based on existing themes.xml).
- Integration with Firebase for dynamic data (Courses, Forum).
