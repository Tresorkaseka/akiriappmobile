# Project Brief - Akiriapp

## Project Overview
Akiriapp is a mobile educational platform (LMS) designed to provide courses, tutorials, and guides to university students. The goal is to create a centralized, modern, and high-quality resource for learning.

## Core Goals
1. Provide a premium learning experience.
2. Centralize course materials (Development, Design, etc.).
3. Facilitate community interaction (Forum).
4. Modern UI/UX with dark mode and vibrant accents.

## Technical Stack
- **Platform**: Android (Native/Kotlin) - based on file extensions seen in metadata.
- **Backend**: Firebase.
- **Design Tool**: Google Stitch for UI prototyping.

## Key Features (Phase 1)
- User Authentication (Login/SignUp).
- Home Page with course discovery.
- Course categories and sections (Popular, Recommended).
- User Profiles and Settings.
- Forum for student interaction.
